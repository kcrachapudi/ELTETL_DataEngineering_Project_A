A fully working ETL pipeline that extracts from REST APIs, CSV, XML, EDI, and fixed-width files — parses all of them into clean DataFrames through a plugin architecture — loads into PostgreSQL with upsert idempotency — exposes an inbound FastAPI server for partner REST and webhook ingestion — fires outbound webhooks, HTTP API calls, and EDI files to partners — with HMAC signing, retry queues, dead letter handling, and a full audit log on everything that moves.
